New davefeedread package. Back when I was doing all my server work in Frontier, I had a verb that, given a URL, would parse a feed and return a struct. It did all the work behind a simple interface. When it came time to fix encoding in feedBase, I decided to invest in having it be that simple in Node. I have the new package running inside feedBase now. It seems to work. You're welcome to use it in your projects as well, of course. #
